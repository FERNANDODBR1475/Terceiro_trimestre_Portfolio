Função Modular:

Denomina-se função modular a função f, de ℝ 𝑒𝑚 ℝ, tal que f(x) = |x|

Uma função módulo é uma função que associa cada elemento de um conjunto a um número real positivo. Esse número real positivo é o valor absoluto do elemento, ou seja, sua distância da origem.

A função modular é representada pelo símbolo |x|, onde x é um número real. Por exemplo: |3| = 3, |-3| = 3, |0| = 0. A função do módulo possui as seguintes propriedades:

|x| ≥ 0, para qualquer número real x. |x| = 0, se e somente se x = 0. |x| = |-x|, para qualquer número real x. |x + y| ≤ |x| + |y|, para todos os números reais x e y. A função módulo é uma função importante em matemática porque é usada em diversas aplicações, como:

Calcule a distância.

Calcule intervalos.

Calcule a raiz da equação.

Calcule funções periódicas.

Aqui estão alguns exemplos de uso do recurso do módulo:

Para calcular a distância de um ponto a ponto (0, 0), podemos usar a função módulo: distância = |(x, y) - (0, 0)|

Para calcular o escopo de uma função, podemos usar uma função de módulo: intervalo = [f(min), f(máx)]

PA e PG

Sequências numéricas são sucessões de números que seguem um padrão específico. Elas podem ser finitas ou infinitas. Para a Matemática, interessa aquelas sequências em que é possível estabelecer uma lei de formação e, assim, descobrir os demais termos.

Chama-se Progressão Aritmética (PA) a sequência de números em que cada termo, a partir do segundo, é a soma do anterior com uma constante dada 𝑟, chamada razão.

Progressão Aritmética (PA) é a sequência de números na qual a diferença entre cada termo (a partir do segundo) e o termo anterior é constante e representado por 𝑟.

Crescente: cada termo é maior do que o anterior, ou seja, 𝑟 > 0.

o Constantes: cada termo é igual ao anterior, então, 𝑟 = 0.

o Decrescente: cada termo é menor que o anterior, logo, 𝑟 < 0.
